# Rock Paper Scissors

A simple interactive Rock-Paper-Scissors game built using HTML, CSS, and JavaScript. Play against the computer and track your score!

## Features

- **Interactive Gameplay**: Click on Rock, Paper, or Scissors images to make your move.
- **Computer Opponent**: The computer randomly selects its move for each round.
- **Score Tracking**: Real-time score updates for both the player and the computer.
- **Dynamic Feedback**: Visual feedback (background color and text) indicates whether you won, lost, or tied the round.

## How to Play

1. Clone or download this repository.
2. Open the `index.html` file in your web browser.
3. Click on one of the three options: Rock, Paper, or Scissors.
4. The game will instantly display the result of the round and update the score.

## Technologies Used

- **HTML5**: For the structure of the game.
- **CSS3**: For styling and layout.
- **JavaScript**: For game logic and DOM manipulation.
